# 🏫 學生教育雲帳號更新工具

🌐 **線上使用**：https://cagoooo.github.io/edu-cloud-updater/

桃園市龍潭區石門國民小學的學生教育雲帳號自動比對工具。每學期從教育局下載最新學生名冊後，自動比對轉入／轉出學生，並產出更新後的完整帳密總表。

## ✨ 特色

- 🔒 **純前端處理** — 所有資料只在瀏覽器本機運算，學生個資不會上傳到任何伺服器
- 🔍 **自動比對** — 以「姓名＋身分證末四碼」為主鍵，自動找出轉出（移除）與轉入（新增）的學生
- 🚨 **末四碼碰撞偵測** — 產出前掃描全校，任何兩位學生末四碼相同（教育雲帳號會撞號）立即紅字警告
- 🔄 **跨學年同步** — 可依新名冊更新在校生的年級／班級／座號（帳號密碼保留原值），暑假升年級、重新編班也適用
- ⚠️ **疑似改名偵測** — 末四碼與生日相同但姓名不同時，自動標示提醒人工確認
- 📋 **智慧欄位辨識** — 自動辨識教育局名冊格式（證照號碼自動取末四碼、出生日期自動轉西元 8 碼）
- 📊 **完整輸出** — 六個年級工作表、依班級→座號排序、標題列凍結窗格、末四碼前導零保留

## 🚀 使用方式

1. 開啟網頁，拖入**現有帳密總表**（.xls / .xlsx）
2. 拖入**教育局新名冊**（學生資料概況_xxxx.xls）
3. 檢視比對結果 → 點擊下載更新後總表（.xlsx）

下學期更新時，直接把上次產出的 .xlsx 當作「現有總表」拖入即可。

## 🛠️ 技術

單一 HTML 檔案，零後端。使用 [SheetJS](https://sheetjs.com/) 讀取 Excel、[ExcelJS](https://github.com/exceljs/exceljs) 產出含凍結窗格的 .xlsx。
